# Recipe · 30 章（单卷单幕）

> 体量：1 卷 × 30 章 × 每章 ≥ 5000 字 ≈ 15–18 万字
> 适合：单弧复仇 / 起源故事 / 悬疑封闭盒 / 校园青春短篇

## 拓扑

- `volumes = 1`
- `acts = 1`（单幕，不需要 act-plan.md）
- `supporting_cast ≤ 3`
- `antagonist_forces = 1`（单股势力即可）
- `conflict_phases`：`survival → (political_intrigue) → (betrayal) → internal_reckoning` 的浓缩

## Chapter Phase 分布

| Ch | 比例 | phase |
|----|------|-------|
| 1–4 | 0–13% | hook |
| 5–10 | 13–33% | setup |
| 11–16 | 33–53% | escalation |
| 17–22 | 53–73% | twist |
| 23–26 | 73–87% | climax |
| 27–30 | 87–100% | resolution_hook |

## Win/Loss 节奏（单卷变体）

- Ch 1：**win**（主角幸存 / 初始目标达成）
- Ch 2–10：多为 neutral + 少量 win（铺垫期）
- Ch 11–22：中段多败（2/3 章是 loss 或 neutral）
- Ch 27：**major loss**（师父陨落 / 挚友死 / 主线挫败）
- Ch 30：**终胜**（闭环 + 开放尾）

## 必做场景清单（obligatory scenes）

| 类型 | Ch |
|------|----|
| 意气觉醒 / 灭门闪回 | 1 |
| 初入门派 / 新环境 | 5 |
| 测骨 / 筛选 | 6 |
| 首次突破（带代价账）| 10 |
| tournament / 小比 | 14–16 |
| 真凶登场 | 18 |
| 卧底揭破 / 朋友反转 | 20 |
| 师门劫难 | 27 |
| 道心抉择 | 29 |
| 名实归位 / 终章闭环 | 30 |

## 伏笔密度

- 至少 **12–15 条** clue → payoff
- Ch 1–8 种植高峰期；Ch 14–22 偿付高峰期
- **全部伏笔必须在 Ch 30 前偿付**

## 快照 / 审计

- character snapshots：Ch 10, 20, 30
- consistency audit：Ch 20（强制）
- rolling summary：**不需要**（< 300 章）

## 推进节奏建议（单轮产出上限）

30 章全量写完需多轮。按 3–5 章一批：

| 批次 | 章节 | 重点 |
|------|------|------|
| 0 | story-bible + volume README | 规划 |
| 1 | ch 1–3 | hook 段 |
| 2 | ch 4–6 | hook → setup 过渡 |
| 3 | ch 7–10 | setup + 首次突破（ch 10 climax）|
| 4 | ch 11–14 | escalation 前段 |
| 5 | ch 15–18 | tournament + 真凶登场 |
| 6 | ch 19–22 | twist 带 ch 20 情感高峰 |
| 7 | ch 23–26 | climax 备战 |
| 8 | ch 27–30 | 终局 |

每批结束同步更新 meta.yaml / volume README / canon facts。
