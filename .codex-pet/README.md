# Codex Pet

This is a tiny local virtual pet for this workspace. It keeps state in
`pet.json` and uses only the Python standard library.

Commands:

```bash
python3 .codex-pet/pet.py status
python3 .codex-pet/pet.py feed
python3 .codex-pet/pet.py play
python3 .codex-pet/pet.py nap
python3 .codex-pet/pet.py rename NewName
```
