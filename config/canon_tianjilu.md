### 铁律

1. **陈天命是陈机的父亲**，不是旁系、不是魔道主帅。全书133章明确"父亲+陈天命"共现
2. 陈天命死于**18年前**。"三年前灵根枯竭"是独立事件
3. **陈玄是300年前历史人物**，第一任天机录主人，非陈机父亲。书中"陈玄"仅10处，均为历史提及
4. **厉青冥才是魔道主帅**，陈天命曾救过他
5. 三兄妹排序：**陈机(大哥) > 陈安(二哥) > 陈念(妹妹)**。"二哥"=陈安是正确的
6. 统一用"陈安"，废弃"陈年"(人名)；"陈年旧事"等成语用法保留
7. 墨先生寄宿于天机录，**非实体**，不能"推门而入"
8. 凌渊是围杀陈天命的参与者，觊觎天机录的竞争者
9. 韩烈是**玄武宗**弟子，不应整日在天青宗内部

## 二、时间线正典

```
十八年前: 陈天命被围杀，对外称"失踪"/"意外"
- 陈机被救出(灭门/大火之夜)
- 陈念在此后不久被圣宗带走(叙事中常与灭门合并叙述)
- 天机录残页封入陈机体内
- 墨先生承诺守护

十一年前: 陈念被圣宗带走(独立时间点，部分章节与灭门同夜叙述为叙事简化)

十年前: 苏青瑶父亲死于"意外"(独立事件，非陈天命时间线)

10年前: 陈机10岁入天青宗外门，沦为杂役

7年前: 陈机13岁，韩烈入玄武宗

3年前: 陈机15岁，灵根"彻底枯竭"，被赶到柴房

第1章: 陈机17岁，天机录觉醒
- 对父亲认知: "失踪多年，生死不明" (正确，不是bug)
- 对陈念认知: 有个失散妹妹，不知下落
- 对陈安认知: 完全不知此人存在

vol1末: 宗门大比，黑马崛起
vol2: 得知"父亲已死"真相 (信息揭露节点)
vol3-4: 秘境争锋，枷锁警示
vol5: 必死劫数，假死脱身
vol6-7: 天眼降临，陈念正式回归
vol8: 命运终章，身份公开
vol9: 天界弈局，陈念牺牲
vol10: 天命归一，打破天道之眼
```

### 时间线铁律

1. 陈天命(父亲)消失/死亡 = **十八年前**。任何"X年前+父亲失踪/消失/死/灭门/围杀/陨落" = 十八年前
2. 陈念被圣宗带走 = **十一年前**。任何"X年前+陈念失踪/被带走/掳走" = 十一年前
3. 苏青瑶父亲之死 = **十年前**（独立事件，不修改）
4. 陈机入宗 = **10年前**
5. "三年前灵根枯竭" ≠ "三年前父亲死了"
6. 三十年前/七十年前/二十三年前等引用陈天命 = 更早的独立事件(凌渊上位/陈天命来皇都等)，不修改
