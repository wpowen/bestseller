### 铁律

1. 陈玄是陈机的**父亲**，不是师父。"师父陈玄"全书均为bug
2. 陈玄死于**18年前**。"三年前灵根枯竭"是独立事件
3. 陈天命与陈机**无血缘**，不是兄长
4. 陈安是**三弟**，不是二哥；统一用"陈安"，废弃"陈年"
5. 墨先生寄宿于天机录，**非实体**，不能"推门而入"
6. 凌渊是觊觎天机录的失败竞争者，**非好友**
7. 韩烈是**玄武宗**弟子，不应整日在天青宗内部

## 二、时间线正典

```
18年前: 陈玄被天道之眼追杀，对外称"失踪"
  - 将陈机(5岁)托付给中间人，安排10岁后入天青宗
  - 将陈念(婴儿)暗中送予圣宗(实为保护)
  - 将陈安(婴儿)送予神秘势力
  - 天机录残页封入陈机体内，设18年后觉醒禁制
  - 墨先生承诺守护

10年前: 陈机10岁入天青宗外门，沦为杂役

7年前:  陈机13岁，韩烈入玄武宗

3年前:  陈机15岁，灵根"彻底枯竭"，被赶到柴房

第1章:  陈机17岁，天机录觉醒
  - 对父亲认知: "失踪多年，生死不明" (正确，不是bug)
  - 对陈念认知: 有个失散妹妹，不知下落
  - 对陈安认知: 完全不知此人存在

vol1末: 宗门大比，黑马崛起
vol2:   得知"父亲已死"真相 (信息揭露节点)
vol3-4: 秘境争锋，枷锁警示
vol5:   必死劫数，假死脱身
vol6-7: 天眼降临，陈念正式回归
vol8:   命运终章，身份公开
vol9:   天界弈局，陈念牺牲
vol10:  天命归一，打破天道之眼
```

### 铁律

1. 陈机入宗是**10年前**，不是5年/3年
2. 第1章陈机不知道父亲已死，只知道"失踪"
3. "三年前灵根枯竭"不等于"三年前父亲死了"
