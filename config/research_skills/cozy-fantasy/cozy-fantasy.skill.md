# 种田仙侠/魔法客栈/田园奇幻调研方法论

调研种田/田园奇幻题材时，以 **生活质感 → 小目标满足 → 生态系统活性 → 稳定中的危机 → 归属感建立** 五层框架推进。

## 1. 生活世界观（world_settings）

**目标**：建立 ≥ 5 种"修仙农庄/客栈的具体生活运转逻辑"，有真实的内在规律。

- 农业仙侠的生态设定：
  - 灵脉等级影响作物品质（但不是线性：高灵脉反而使部分灵植"躁动"）
  - 气候/节气 × 灵植生长周期 = 季节性叙事节奏
  - 灵兽与灵植的共生关系（驱虫、授粉、感知天气）
- 客栈/经营世界：
  - 修士路途中的需求（休息、情报、特殊食材、储物空间）
  - 定价、声望、特色服务——不是平等的，有层次
- `content_json` 须含：`livelihood_logic / seasonal_rhythm / ecosystem_rules / economic_model`

## 2. 主角设计（character_archetypes）

**种田主角必须有"真正在乎这里"的情感锚点，不是因为"没处去才种田"。**

- 在乎什么？（至少选 2 个）：
  - 一块特定的土地或植物
  - 某个固定出现的顾客/邻居
  - 某种手艺（料理/炼制/药圃）
  - 对"慢生活"本身的珍视（有哲学支撑，不是懒惰）
- 隐藏实力 ≠ 无限碾压：高人出手应该有代价（体力、暴露身份、打破平衡）

## 3. 情节节奏（plot_patterns）

**生活三件事结构 + 季节性推进 + 每 8-12 章一个"真正的危机"。**

- 日常节奏：
  1. 小目标（收获某作物 / 招待特殊客人 / 培育稀有品种）
  2. 小障碍（天气变化 / 竞争者 / 灵兽失控）
  3. 小成就（成功 + 一个意外收获）
- 季节性：春种 → 夏管 → 秋收 → 冬藏，每季节对应情节章节密度
- 外部危机处理：必须有，但规模控制在"生活层面"，不升级到末日级别

## 4. 场景与感官（scene_templates + locale_templates）

- 种植/收获场景的感官细节：不只是"灵植成熟了"——要有味道/颜色/光泽/触感
- 料理场景：从食材处理到成品，每步有灵气转化的叙事意义
- 地点多层次：
  - 农庄本体（主角主权范围，安全）
  - 周边秘境（探索范围，有风险但规模可控）
  - 远处世界（偶然出现的消息/访客带来，不直接卷入）

## 5. 温情叙事节奏（emotion_arcs）

- 不要从第一章就"选择归隐"——最好有一个"意外来到这里、慢慢爱上"的转变弧
- 配角们的故事：每个常客 / 邻居要有自己的小弧线，主角只是其故事的一个角色
- CP推进：不用大起伏，靠共同生活细节积累——共享一顿饭比表白更有说服力

## 6. 真实参考（real_world_references）

- **二十四节气**：提供天然的章节节奏参考；不同节气对应作物行为、天气、宜忌
- **中草药文化**：《神农本草经》里的药材，有真实的生长习性、效用、稀有度层次
- **里山里海理念**（日本）：人与自然半共生状态，边界模糊，不是占有也不是放弃
- **游戏设计**：《星露谷物语》的任务循环（小目标闭环），《动物森友会》的无压力节奏设计
