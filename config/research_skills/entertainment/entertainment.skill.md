# 娱乐圈甜宠/选秀/直播调研方法论

调研娱乐圈题材时，以 **行业生态 → 资本结构 → 人设管理 → 情感真实性 → 成功代价** 推进。

## 1. 世界观（world_settings）

**目标**：建立 ≥ 5 种"内娱权力结构"模型，每种呈现不同的资本与内容的关系。

- 探索：
  - 流量主导型（数据为王，无流量无资源）
  - 实力优先型（奖项与作品口碑驱动）
  - 资本绑定型（某大公司的存在改变整个生态）
  - 直播/短视频新生态（平台打破传统渠道）
- 每条要有：`power_holders / entry_barriers / success_metrics / manipulation_patterns`。

## 2. 人物（character_archetypes）

- 女主必须有专业能力（不是单靠颜值和男主保护）。
- 男主顶流的孤独要有现实依据：舆论撕扯/行业竞争/角色认知错位。
- 反派资本方：有合理商业逻辑，不是单纯"坏人"。

## 3. 情节（plot_patterns）

- 不要用"天赋一击定胜负"——成功必须包含：积累 + 机遇 + 代价。
- 暗线推荐：行业内幕揭露 / 真实身份隐藏 / 过去事件的影响。
- 恋情公开必须有危机：粉丝反应、经纪公司压力、合约限制中选至少一条。

## 4. 对话（dialogue_styles）

- 公开场合的公关语言 + 私下的真实语言要有落差感。
- 粉丝留言/弹幕文化要有真实的互联网语感。
