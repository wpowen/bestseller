# 史诗奇幻/大陆争霸/世界树体系调研方法论

调研史诗奇幻题材时，以 **宇宙论底层 → 多阵营张力 → 史诗节奏管理 → 个人与宏观交织 → 代价与母题深度** 五层框架推进。

## 1. 宇宙论与世界史（world_settings）

**目标**：建立有真实历史感的世界——读者应该感到"这个世界在主角出生前几千年就在运转了"。

- 神话层（创世/末世/神祗干预机制）
- 历史层（帝国兴衰/大战记录/文明断层）
- 现实层（当前政治格局/主角所在时代的意义）
- 三层之间必须有**因果连接**，不是三个独立设定堆叠
- `content_json` 须含：`creation_myth / historical_eras / current_tension / divine_mechanism`

## 2. 多阵营设计（factions）

**史诗奇幻的张力来自"每一方都有合理性"，不是正义 vs 邪恶。**

- 每个阵营需要：
  - 内部逻辑（他们为什么相信自己是对的）
  - 资源基础（他们靠什么生存/扩张）
  - 内部裂痕（阵营内部也有派别和分歧）
  - 与其他阵营的利益纠葛（不是"天生敌人"）
- 种族设计原则：不同种族的**价值观差异**比外观更重要（什么是荣耀？什么是罪？什么是死亡？）

## 3. 史诗叙事节奏（plot_patterns）

**宏观战争 + 个人故事 = 双轨道叙事，不能只有其一。**

| 层面 | 功能 | 节奏密度 |
|------|------|---------|
| 宏观战役/政治 | 给读者世界感 | 每 3-4 章一个宏观事件 |
| 中观阵营动态 | 张力来源 | 每章至少有信息更新 |
| 微观个人故事 | 情感投入 | 每章的主体内容 |

- 高潮战役前：必须有足够的"平静积累期"，让读者在乎的东西多到足以被毁灭
- 命运预言：可以用，但最终必须"以非预期方式成真"——不是字面实现

## 4. 神话母题（thematic_motifs）

- **代价叙事**：史诗英雄的胜利必须付出不可弥补的代价——这是史诗与爽文的核心区别
- **秩序与混沌**：不是好与坏，是两种宇宙观的碰撞（秩序的保守/压制 vs 混沌的创造/毁灭）
- **英雄之殇**：最强大的英雄往往最孤独——史诗的悲剧底色

## 5. 真实神话参考

- **北欧神话**：九界宇宙论、诸神黄昏（末世论）、奥丁的代价哲学（知识需要牺牲）
- **希腊神话**：英雄 + 缺陷（阿喀琉斯之踵）、神祗任性干预凡人事务
- **中国上古神话**：《山海经》的异兽异族体系、混沌创世与人文起源的融合
- **真实帝国历史**：罗马/蒙古帝国的扩张逻辑，给"大陆争霸"提供政治经济真实感
