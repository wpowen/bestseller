# 大女主/宫斗权谋/后宫调研方法论

调研宫斗/大女主题材时，以 **权力场域 → 女性边界 → 谋略设计 → 情感与权力张力 → 代价呈现** 推进。

## 1. 权力场域（world_settings + factions）

**目标**：建立 ≥ 6 种"后宫/宫廷权力结构"，明确女性可以合法触达的权力资源。

- 女性可以使用的权力工具：
  - 子嗣（最传统，最不稳定）
  - 家族背书（外戚势力）
  - 宫廷人脉（太监、女官、宫女网络）
  - 信息控制（情报、秘密、把柄）
  - 皇帝情感（宠爱，但必须主动维系）
- 每种权力工具有其成本和脆弱点，不是单向优势。

## 2. 人物（character_archetypes）

- 大女主的成长**必须付出真实代价**：失去友情、放弃情感、背叛价值观中至少一条是必要情节。
- 女配的对立必须有合理动机，不是"嫉妒"标签——她们在争夺同样的稀缺资源（皇帝、爵位、安全感）。
- 男性角色不应是主角的工具：皇帝/男主有自己的政治逻辑。

## 3. 情节（plot_patterns）

- 谋略要有**三步推演**：步骤 A 导致状态 B，B 触发 C 的反应，C 被主角利用完成目标——中间不能跳步。
- 政治阴谋的信息量要分散在多章，不能一章交代清楚。
- 高光时刻必须是主角主动决策的结果，不是运气或男主救场。

## 4. 对话（dialogue_styles）

- 宫廷对话必须有多层含义：表层说的是A，实际传达B，外人不懂C层。
- 女主的谋略要在对话里体现，而不是独白告诉读者"她已经想好了"。
