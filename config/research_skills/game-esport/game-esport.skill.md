# 游戏电竞/无限流/副本流调研方法论

调研游戏化题材时，以 **规则系统 → 目标结构 → 竞争关系 → 成长代价 → 真实感锚点** 推进。

## 1. 规则世界（world_settings）

**目标**：≥ 8 种"进入游戏化世界的机制"，每种机制对玩家与现实的关系处理不同。

- 探索：
  - 意识传送型（肉体留原世界）
  - 完全转移型（原世界等效时间流速不同）
  - 层叠型（无限流：随机场景切换）
  - 混合型（游戏规则入侵现实）
  - 单一副本型（玩家之间互相对抗的空间）
- 必须明确：死亡惩罚机制 / 是否存在读档 / 外部时间流速。

## 2. 能力与进阶（power_systems）

- 避免复制电子游戏的力量/敏捷/智力三围——这已经过饱和。
- 探索差异化体系：
  - 概念型（掌握"火"的概念，而非"火系法术"）
  - 规则改写型（修改副本内局部规则）
  - 社交型（影响 NPC 和玩家关系网络的能力）
  - 信息型（知道别人不知道的副本信息）
- 每条 entry 须含：`acquisition_cost / ceiling / societal_impact / counter_mechanism`。

## 3. 组织（factions）

- 宗门经营类特别注意：内部矛盾比外部竞争更有戏剧张力。
  - 新老弟子的资源分配争夺
  - 宗门方向路线之争（战斗派 vs 修炼派）
  - 外来高手冲击内部权力格局

## 4. 情节（plot_patterns）

- 无限流：每个副本要有独立的"世界逻辑"，不能只是换皮打怪。
- 非战斗副本至少占比 30%：谈判、生存、解谜、表演、经营类。
- 宗门经营：核心矛盾应是"资源不够用时谁被牺牲"，不是"资源越来越多"。

## 5. 真实感（real_world_references）

- 参考真实电竞生态：训练日程、合同纠纷、赞助关系、心理健康问题。
- 系统类规则借鉴真实桌游机制（Roguelike死亡惩罚、TCG稀缺性、RPG支线可选性）。
