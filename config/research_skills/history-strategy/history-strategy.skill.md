# 历史权谋/穿越争霸调研方法论（v2.0）

调研历史题材时，以 **朝代结构 → 政治势力 → 经济基础 → 人物动机 → 事件逻辑 → 语言质感** 六层推进。

## 1. 世界观与朝代结构（world_settings）

**目标**：提供 ≥ 8 种历史/架空朝代的权力架构模型，每种有独特的宫廷结构、军事制度、经济体系。

- 覆盖范围：
  - 真实朝代（汉、唐、宋、明、清）的权力结构特征
  - 架空变体（单改制度框架，保留地理与文化背景）
  - 非华夏视角（北方游牧政权、南方海上贸易政权）
- 重点标注：
  - 皇权与相权的对立关系（每个朝代不同）
  - 地方势力与中央的张力（如藩镇、土司）
  - 商业集团的政治介入方式（徽商与官场的共生）
- `content_json` 须含：`dynasty_type / power_balance / economic_base / factional_structure / information_control`

## 2. 派系（factions）

**权谋不止发生在宫廷——必须覆盖全体系。**

- 文官派（科举出身，礼法为器，清流 vs 浊流）
- 武将派（边疆军功，皇帝制衡，忠君 vs 割据）
- 宦官集团（皇权延伸，也是威胁，有自己的经济利益）
- 商帮（货币权力，隐形政治力，官商界面的代理人）
- 秘密结社（白莲教、天地会类原型，有意识形态支撑）
- 外来势力（使节/贸易国/内附民族，有独立利益诉求）

每个 entry 须含：`core_interest / patron_relation / vulnerability / long_term_goal / distinctive_resource`

## 3. 人物（character_archetypes）

**权谋文的人物设计必须有"政治逻辑"作为驱动，而不是情感驱动。**

- 谋士类型多样化：
  - 以退为进型（通过失去权位来保命再布局）
  - 多方下注型（同时服务多个主公，忠诚度动态）
  - 隐者入局型（世外高人被动介入，但有自己的目标）
  - 技术性谋士（精通某个领域的专才，非全能天才）
- 女性政治人物：
  - 不止皇后和公主
  - 商人、文人妻、女医、女将——各有不同的权力接触路径
- 穿越主角的知识局限：
  - 他/她的历史知识也有盲区（哪段历史没读过？）
  - 现代人在古代的适应性弱点（体力/情绪/语言）

## 4. 情节设计（plot_patterns）

**明线 + 暗线的黄金组合：**

| 明线 | 暗线 | 代价类型 |
|------|------|---------|
| 夺位/争霸 | 身世秘密 | 失去一段真情 |
| 复仇 | 前朝阴谋 | 变成自己最恨的那种人 |
| 经商/立业 | 政治阴谋 | 商业成功但失去家人 |
| 开疆 | 情感代价 | 获得功业但孤老而终 |

- 信息差博弈：A知道B不知道 → B知道A不知道A知道 → 信息层次叠加
- 权谋节奏：谋划期（慢节奏）+ 执行期（快节奏）+ 代价期（情感慢节奏）

## 5. 对话质感（dialogue_styles）

**古代对话的层次感是区别优劣权谋文的关键。**

- 三层含义：
  1. 表面：说的是什么（礼仪化的内容）
  2. 实际：传达了什么（权力信号）
  3. 隐藏：没说的是什么（双方都知道的潜规则）
- 朝堂奏对：有文言感但不能让读者读不懂——半文言，关键词文言，叙述白话
- 商业谈判：间接语言，不直接报价，通过行动和场景暗示价码

## 6. 真实史学参考（real_world_references）

- **《战国策》**：纵横家说辞的实际案例——如何用语言扭转局势
- **明代宦官政治**：魏忠贤等案例，权力来源不在官职，在信息控制
- **徽商与官场**：商业资本如何渗透政治，有具体操作逻辑
- **二十四史中的权谋案例**：史记·商鞅变法、汉书·王莽篡位，每个都有完整的权谋逻辑链
