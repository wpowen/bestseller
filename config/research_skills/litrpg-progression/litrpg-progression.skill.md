# LitRPG/炼丹炼器/数值成长调研方法论

调研制造/数值题材时，以 **稀缺性设计 → 制造循环 → 成长代价 → 工匠人格 → 经济生态** 五层框架推进。

## 1. 世界观与经济生态（world_settings）

**目标**：建立 ≥ 5 种"炼制/数值体系影响整个社会的方式"，不仅仅是战力层面。

- 材料/资源的稀缺逻辑：
  - 产地限制（某矿脉/某秘境/季节性采集）
  - 材料配伍制约（某 A+B 才能触发反应，单独无效）
  - 等级/成功率天花板（五阶炼师炼制六阶器出品率 < 5%）
- 社会分工：
  - 炼师/丹师阶级 vs 战斗修士阶级的资源依赖与博弈
  - 炼制行业的行会、认证、价格卡特尔
  - 顶级工艺的"传承断层"——秘方失传 / 炉鼎失毁
- `content_json` 须含：`scarcity_model / production_chain / social_impact / bottleneck_resource`

## 2. 数值体系设计（power_systems）

**数值体系必须有内生的"边际效应"——不能无限线性成长。**

| 元素 | 最低要求 |
|------|----------|
| 境界层级 | 每层有具体能力变化，不只是"更强" |
| 突破壁垒 | 必须有非纯堆材料的障碍（顿悟/心境/材料稀缺） |
| 数值上限 | 存在客观天花板或收益递减 |
| 副作用 | 高级炼制有副作用/风险（身体耗损/精神损耗/材料爆炸） |

- 炼丹：丹方 × 火候 × 炉鼎 × 修士本身感知 = 多变量系统
- 炼器：材料灵性 × 铭文适配 × 锻造意志注入 × 淬炼次数 = 层次感
- 符修：符文结构 × 能量容量 × 触发条件 = 组合数爆炸

## 3. 主角人格设计（character_archetypes）

**工匠型主角的差异化竞争力不是"战斗能力"，而是"理解物"的能力。**

- 核心区别：战斗型主角的世界观是"强者的逻辑"；工匠型主角的世界观是"物理/材料/规律的逻辑"
- 成长的本质不是变更强，而是"对材料的理解更深"
- 内在哲学锚点：
  - 制造行为本身的意义（不是为了战斗而造，而是为了"让物存在"）
  - 与材料的"关系"（不同于工具论，接近日本工匠精神）
  - 创新 vs 传统：打破旧丹方 vs 守护传承的张力

## 4. 情节节奏（plot_patterns）

**制造循环是节奏引擎，不是背景板。**

炼制章节的三段式结构：
1. **准备段**：材料获取/筛选（含挑战和选择），建立期待
2. **过程段**：炼制中的变数（意外/危机/顿悟），维持张力
3. **结果段**：成品呈现 + 真实后果（高阶成品引发外部注目/竞争）

- 每隔 10–15 章要有一次"炼制彻底失败"的情节，且失败有真实代价
- 突破节点不只靠炼制——境界突破需要"技术 + 心境 + 机缘"三重条件

## 5. 系统/数值面板的文学化处理（dialogue_styles）

- 系统提示信息：每章 ≤ 2 次系统面板展示，且每次必须与情节直接相关
- 数字不等于意义：不要让主角因为 "Lv +1" 就高兴——需要用角色的感受/行为表达成长
- 工匠专业术语：适度使用，但每个专有名词初次出现必须有情境化解释（读者不能靠百科才懂）

## 6. 真实参考（real_world_references）

- 中国道教外丹传统：炼丹术历史（汉代 → 唐代鼎盛 → 宋代式微），炼制步骤"采药 → 精制 → 入炉 → 炼制 → 出丹"
- 日本刀匠工艺：玉钢（tamahagane）、折叠锻造（折り返し鍛錬）的仪式感与精确性
- 游戏设计参考：《我的世界》合成系统（组合爆炸）、《魔兽世界》工程学/附魔（社会分工）
- 经济学：萨伊定律 vs 凯恩斯需求端，在虚构经济体里的炼制供需应用
