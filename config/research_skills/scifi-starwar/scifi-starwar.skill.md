# 星际科幻/星战/机甲调研方法论（v2.0）

## 1. 世界观（物理基础要真实）

**明确星际物理约束：**
- 超光速通讯是否存在？代价是什么？
- 超光速航行是否存在？如何实现？（虫洞/曲速/跳跃点/冬眠）
- 通讯延迟 → 去中心化必然 → 星际帝国行政现实（不能一人管所有星球）

**社会结构差异化：**
- 被外星文明殖民后的人类社会
- 多个殖民星发展出的独立文化支流
- 人类分化成多种亚种（太空适应/基因改造/义体融合）
- `content_json` 须含：`stellar_geography / ftl_model / communication_lag / social_divergence`

## 2. 体系（power_systems）

修炼体系可置换为科技等级框架：
- 科技等级 / 机甲驾驶熟练度 / 脑机接口稳定度 / AI融合度
- 禁止只用"机甲驾驶员能力 = 神经反应速度"单轴指标
- 技术有副作用：脑机接口的认知损伤，基因改造的心理副作用

## 3. 派系（factions）

星际帝国 vs 星际联邦可扩展为：
- 星际宗教势力（以信仰为组织原则的跨星文明）
- 星际企业巨头（不受国家约束的跨星际经济体）
- 殖民星独立军（反抗地球/核心星的边缘力量）
- 未被承认的AI文明（意识是否构成公民权？）
- 外星族群（非人类逻辑的政治体制）

每个势力须含：`economy / governance / military_doctrine / cultural_profile / core_vulnerability`

## 4. 人物（character_archetypes）

**禁用"天才驾驶员 + 银河帝国皇子"。**

鼓励非战斗核心职业：
- 后勤参谋（真实的星际战争决定于补给，不是战场）
- 星际外交官（文化翻译员，不是武器使用者）
- 星舰工程师（硬科幻的技术接口）
- 殖民地农学家（星际扩张的底层）
- 战后心理医生（星际PTSD的叙事价值）

## 5. 场景设计（scene_templates）

- 星舰对战：不要"齐射—中弹—爆炸"三段式
  - 真实海军战术参考：侦查→情报战→电子干扰→补给线切断→心理战→决战
- 机甲驾驶舱特写：避免"全息屏+红色警报+驾驶员冷汗"
  - 改为：物理感官（震动/过载/温度）+心理感受（恐惧/专注状态）+战术决策

## 6. 真实科学参考（real_world_references）

- **相对论**：时间膨胀意味着星际旅行后的"时间流放"（出发时老的人可能比主角更年轻）
- **卡尔达舍夫指数**：文明能源等级，给文明分级提供有科学依据的框架
- **费米悖论**：宇宙沉寂的原因，提供"大过滤器"作为世界观底层压力
- **国际空间站运营**：真实的太空生活细节（物资限制/心理压力/应急程序）
