# 社畜摆烂/沙雕种田/轻喜剧调研方法论

调研轻喜剧/摆烂题材时，以 **轻松但有温度 → 日常中有成长 → 低门槛冲突 → 情感小满足** 四层框架推进。

## 1. 世界观（world_settings）

**目标**：建立 ≥ 5 种"低压力生活场景"的世界设定，每种有具体的"为什么在这里比现代更放松"的逻辑。

- 探索：
  - 异世界田园（魔法种田，规则简单）
  - 小镇经营（酒馆/面包店/旅店，固定客源）
  - 修仙养老（无人知道是老前辈，安心摆烂）
  - 现代咸鱼（都市里的低欲望区域：小城、返乡、社区）
- 重要：世界要有真实的"小麻烦"来源，不是无压力——麻烦必须是生活规模，不是末日级别。

## 2. 人物（character_archetypes）

- 摆烂主角**必须有一个真实的东西在乎**：友谊、食物、小动物、某个传统——这是情感锚点。
- "隐藏的强大"要晚些展示，且代价是必须暂时放弃摆烂。
- 配角们的困扰要有轻松解决路径，主角的帮助不应该是武力/金钱。

## 3. 情节节奏（plot_patterns）

- 日常三件事结构：小目标 → 小障碍 → 小成就，每章完结感强。
- 每 5-10 章要有一次"被迫认真"的情节，让读者看到主角的真实能力。
- 结局前必须有一次"主角真的要失去某样珍视的东西"的危机。

## 4. 轻喜剧感（dialogue_styles）

- 包袱要有层次：铺垫 → 误导 → 意外打破 → 主角反应。
- 主角的内心吐槽是核心笑点：和读者共鸣的社会观察。
- 避免强行搞笑：不是一直说冷笑话，是日常情境自然发生的反差。
