# 星际争霸/太空歌剧/科幻帝国调研方法论

调研星际/太空歌剧题材时，以 **宇宙尺度建立 → 文明生态设计 → 星际政治张力 → 个人与文明交织 → 科学哲学底层** 五层框架推进。

## 1. 宇宙尺度感（world_settings）

**目标**：读者应该感受到"这是一个有真实历史感的星际文明"，不是地球故事换了背景板。

- 文明历史分层：
  - 古文明遗迹（先行种族/失落帝国）
  - 扩张时代（殖民战争/第一接触）
  - 当代格局（现有势力均衡或失衡）
- 物理规则落地：
  - FTL 方式决定势力范围（虫洞固定 vs 曲速随行 vs 跳跃点）
  - 通信延迟 → 决策机制（去中心化必然）
  - 资源分布 → 政治格局（稀缺矿星必争）
- `content_json` 须含：`stellar_geography / ftl_model / communication_lag / resource_choke_points`

## 2. 文明层次设计（factions）

**星际文明不是"大一点的国家"，有真正不同的文明逻辑。**

- 物种文化差异的三个维度：
  1. **时间观**（短命种族的急切 vs 长寿种族的耐心）
  2. **集体/个体**（蜂群意识 vs 极端个人主义）
  3. **存在目的**（繁殖扩张型 vs 知识积累型 vs 虚无型）
- 政体类型的叙事意义：
  - 帝国：中央集权效率高，但内部腐化 + 边疆失控
  - 联邦：民主决策慢，但韧性强
  - 企业联合：逐利高效，但无公共利益逻辑

## 3. 星际叙事节奏（plot_patterns）

**宏观战争 + 个人视角 + 科学哲学三轨并行。**

- 宏观层：文明/战争走向（每 5-8 章一次大事件）
- 中观层：主角所在单位/阵营的动态（每章信息更新）
- 个人层：主角的内在成长（每章的主体视角）
- 太空战的叙事节奏不同于地面战：信息战/博弈时间比交战时间长得多，利用好这个特性

## 4. 核心母题（thematic_motifs）

- **文明是否值得延续**：太空歌剧最深层的问题，不是"谁赢"，而是"为什么要活"
- **人类中心主义的终结**：多种族宇宙中，人类价值观未必是普世价值
- **技术与人性**：高科技文明里，人类性是否需要保留，代价是什么

## 5. 真实科学与历史参考

- **卡尔达肖夫指数**：I型（行星级能源）→ II型（恒星级）→ III型（星系级），给文明分级提供框架
- **费米悖论**：宇宙那么大为什么没有外星文明接触——大过滤器假说是绝佳世界观底层
- **罗马帝国行省管理**：行政体系的腐败与边疆失控，是星际帝国的现实参考
- **冷战历史**：意识形态对立下的太空竞赛，提供星际政治的人性真实感
