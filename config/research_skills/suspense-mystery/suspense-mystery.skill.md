# 悬疑推理/怪谈惊悚调研方法论（v2.0）

调研悬疑题材时，依 **谜题结构 → 信息管控 → 人物动机 → 氛围营造 → 规则设定** 推进。

## 1. 世界观与规则（world_settings）

**目标**：建立 ≥ 6 种"限制读者信息的叙事世界"，每种有独立的信息透明度设计。

- 信息控制框架：
  - 第一人称限知型（读者只能知道"我"知道的，错的也跟着错）
  - 多视角拼图型（每个视角有盲区，读者独立拼图）
  - 不可靠叙述者型（叙述者本身有记忆错误/主动欺骗）
  - 规则型恐怖（民俗/怪谈里的禁忌系统："你不能做XX否则…"）
  - 双时间线型（现在调查 vs 过去事件，读者有信息差优势）
- 民俗怪谈的"规则"必须在开头完整设定，结局不能新增规则
- `content_json` 须含：`info_control_method / reader_advantage / mystery_strata / resolution_type`

## 2. 谜题三层结构（plot_patterns）

**目标**：每个谜题都有三层，不是单一问题。

| 层次 | 类型 | 揭示时机 |
|------|------|---------|
| 表层谜 | 可快速推理，信息已在前文 | 中间章节小满足 |
| 中层谜 | 需要积累才能理解，有误导 | 3/4处的大反转 |
| 底层谜 | 揭开改变一切的真相，颠覆前两层 | 结局 |

- 前三章必须埋下至少两个"可疑不合理点"供读者抓住
- 误导线索（red herring）：必须有合理性（不是随机撒，而是凶手/叙述者蓄意布置）

## 3. 人物（character_archetypes）

- 避免"全知天才侦探"——读者更认同有盲点的侦探。
- 探索：
  - 局内人型（凶手或受害者周围的人，无法客观）
  - 被迫介入型（普通人意外卷入，资源和逻辑都不如专业人士）
  - 双重身份型（侦探本身有秘密，调查威胁到自己）
- 反派/凶手动机必须有**社会结构根因**（不是"天生变态"，而是被系统逼到这一步）
- 不可靠叙述者设计：叙述者的谎言必须有**心理动机**，不是作者故意瞒读者

## 4. 民俗怪谈专项（locale_templates + real_world_references）

**中国特色地点选择：**
- 南方水乡：水面反射、船屋、鱼塘——视觉扭曲感
- 矿区：地下空间、封闭、机械噪音——压迫感
- 古村祠堂：族规、秘密、前人遗影——历史压迫
- 精神病院：记忆与现实边界——可靠性挑战

**民俗素材参考：**
- 湖南赶尸：具体操作、禁忌、匠人与亡灵的契约
- 茅山法术：符咒逻辑、禁忌体系、"客观效果"的民间解释
- 广东纸钱/纸扎：祭品与亡魂沟通的物质媒介，叙事道具
- 各地地方志怪谈：如《阅微草堂笔记》、《子不语》——有具体地点和情节

## 5. 氛围营造（scene_templates + dialogue_styles）

- **心理恐怖 > 视觉恐怖**：用角色的反应和诠释制造恐惧，而不是直接描述恐怖事物
- 恐怖场景的"之前和之后"：最恐怖的是"快发生之前"和"刚发生之后"，事件本身可以省略
- 侦探审讯对话：设问 → 观察反应 → 再设问（不是直接指控）——每句话都有信息目的

## 6. 真实参考（real_world_references）

- **FBI 犯罪侧写**：真实凶手动机分析（系统性而非随机），提供有说服力的动机结构参考
- **中国刑侦案件**：真实案件中的信息误导/排查误差，比纯文学案件更有写实感
- **中国考古异常发现**：三星堆/马王堆等出土物的未解之谜，可作为民俗怪谈的"真实锚点"
