# 系统末日/觉醒异能/治愈辅助调研方法论

调研系统末日/治愈主角题材时，以 **非战斗价值体系建立 → 稀缺性设计 → 社会博弈 → 代价与成长 → 叙事差异化** 五层框架推进。

## 1. 末日世界的经济逻辑（world_settings）

**目标**：建立"治愈者在这个世界有真实稀缺价值"的经济/社会生态，不是"因为主角是好人所以大家都爱她"。

- 治愈者稀缺的客观原因：
  - 觉醒概率本就低（每万人中几率）
  - 治愈职业路线特殊，高比例觉醒者放弃（觉醒初期被当废物）
  - 治愈者高死亡率（被争夺/被绑架/前线消耗）
- 供需决定地位：
  - 治愈者 = 稀缺资源 → 不同势力争夺 → 主角的自主性/谈判筹码
  - 无治愈者的团队 vs 有治愈者的团队：生存率差异有多大？（给出具体数字感）
- `content_json` 须含：`healer_rarity / supply_demand_model / power_structure_impact / scarcity_logic`

## 2. 治愈能力体系（power_systems）

**治愈技能不是"加HP"——有多个维度的差异化设计。**

| 技能类型 | 效果 | 代价 | 叙事价值 |
|---------|------|------|---------|
| 即时治愈 | 快速恢复外伤 | 精力消耗高 | 战场必需 |
| 慢性治疗 | 根治内伤/毒素 | 时间长、要求安全环境 | 团队基础 |
| 状态恢复 | 清除异常状态 | 材料消耗 | 稀缺价值 |
| 增强治愈 | 超越自然恢复限制 | 需要特殊条件 | 战略价值 |
| 死亡边缘 | 拉回濒死 | 极高代价，损寿 | 关键情节用 |

- 治愈**必须有代价**：体力/精神力/材料/时间——任何一种都行，但必须存在
- 高级治愈技能的**解锁条件**不只是等级，还要见过相应的伤情（经验积累型）

## 3. 非战斗主角的生存逻辑（character_archetypes）

**辅助型主角的核心竞争力不是战斗，而是：**
1. **信息优势**：治愈者能判断伤势 → 判断敌人的实力/能力类型
2. **资源控制**：稀缺材料/技能 → 谈判筹码
3. **人际网络**：每个她救过的人，都是未来的债权/保护关系
4. **判断力**：见过太多死亡 → 对危险有准确判断，不冲动
- 主角的**底线设计**：她拒绝治疗什么人？在什么情况下拒绝？这比"救所有人"更有叙事价值

## 4. 情节差异化（plot_patterns）

**治愈师主导情节的方式，不靠战斗力：**

- **信息战**：通过治疗获取伤势信息 → 判断敌方实力/战略 → 提前部署
- **资源战**：控制某稀缺材料/技能 → 势力争夺 → 主角的选择决定格局
- **人心战**：建立真实的人际关系 → 关键时刻有人保她/为她战
- **拒绝治疗**：当主角拒绝治疗某人时，是情节高点（权力的另一面）

## 5. 治愈者的创伤积累（emotion_arcs）

- 每次救人成功 ≠ 快乐：见过太多死亡，会有"为什么有些人能救有些人不能"的心理重量
- **选择谁先救**：末日资源紧缺时，治愈者必然面对选择困境——这是最强情感张力
- **不能救的人**：比救人场景更有冲击力——建立真实的代价感

## 6. 真实参考（real_world_references）

- **战场医疗/急诊护理**：真实医护工作的决策压力、情感消耗、物资紧缺
- **MMO游戏治愈职业**：供需关系（找治愈难）、团队博弈（治愈者的话语权）
- **灾难心理学**：幸存者综合症、救人者的心理创伤——治愈师角色的现实参照
- **中世纪战场医生**：有技能但地位低、经常被强征——历史上"有用但脆弱"的真实处境
