# 都市修仙/灵气复苏调研方法论（v2.0）

## 1. 世界观：超越"灵气复苏0年"

库里已经囤积了大量"灵气刚复苏、主角在大街上突然觉醒"的模板（usage > 6），下一条目应该探索：

- **半灵气时代**（只有部分人/部分地区有灵气，边界的社会摩擦）
- **灵气倒退**（曾经高修为，现在只剩痕迹，历史上有过巅峰期）
- **数字灵气**（灵气通过互联网传播/被企业垄断，修炼要账号）
- **无灵气但有异能**（异能来源非灵气，而是基因/情绪/数据）
- **灵气污染**（修炼的副产物，影响普通人，社会矛盾来源）

`content_json` 须含：`urban_layer_model / power_public_awareness / faction_hide_mechanism / social_inequality_mechanism`

## 2. 体系：脱离"奇经八脉"套路（power_systems）

- 契约体系（与都市基础设施签约：电网、水系、地铁——断约即断修）
- 数据体系（异能以数据流形式传递，赛博+仙侠）
- 情绪体系（修为由情绪稳定度决定，内心越乱越危险）
- 职业共鸣型（职业和修炼路线绑定，医生/律师/程序员各有不同修炼路）
- 每条 entry 必含：`levels[] / upgrade_triggers / modern_integration / social_awareness_cost`

## 3. 人物原型（character_archetypes）

**禁用**：外卖员 / 保安 / 低调学生 / 被裁员的小职员开局。

鼓励有真实职业背景的主角：
- 企业高管（掌握资源，但进入修仙体系后的身份焦虑）
- 法医（对死亡的专业认知，与怪谈/异能的接触）
- 刑警（执法者面对异能犯罪，制度框架的撕裂）
- 医生（修仙与现代医学的知识冲突）
- 记者（信息权力，揭露隐藏世界的职业动机）

## 4. 场景设计（scene_templates）

- 觉醒场景**不要**在"上班路上 + 便利店 + 下雨天"；这三要素同时出现 usage 已超过 10 次
- 斗法要有**现代后勤**：监控记录/社交媒体流出/维稳部门介入/医保理赔
- 修炼地点的现代化：写字楼顶层/酒店行政套房/地下停车场——环境反差强化奇幻感

## 5. 都市修仙的社会层（world_settings）

**灵气复苏改变社会结构，这是都市修仙的核心母题：**
- 觉醒者 vs 非觉醒者的阶级分化（原本的社会精英可能觉醒失败）
- 企业资本如何抢先垄断修炼资源
- 政府如何试图管控修仙者（许可证/监控/征召）

## 6. 真实参考（real_world_references）

- **中国气功热（80年代）**：真实的社会现象，全民参与的超自然崇拜，提供"灵气复苏社会反应"的真实参考
- **新宗教研究**：当代人如何接受超自然体验并纳入日常生活逻辑
- **阶层固化研究**：灵气复苏后的不平等可以对应现实阶层研究，增加社会批判深度
