# 都市修仙调研方法论

## 1. 世界观：别只做"灵气复苏 0 年"

库里已经囤积了大量"灵气刚复苏、主角在大街上突然觉醒"的模板（usage > 6），
下一条目应该探索：

- **半灵气时代**（只有部分人 / 部分地区有灵气）
- **灵气倒退**（曾经高修为，现在只剩痕迹）
- **数字灵气**（灵气通过互联网传播 / 被企业垄断）
- **无灵气但有异能**（异能来源非灵气，而是基因 / 情绪 / 数据）

## 2. 体系：脱离"奇经八脉"套路

- 契约体系（与都市基础设施结契约：电网、水系、地铁）
- 契合体系（与特定职业 / 建筑 / 器物契合）
- 数据体系（异能以"数据流"形式传递，类似赛博 + 仙侠）
- 情绪体系（修为由情绪稳定度决定）

## 3. 人物原型

- **禁用**：外卖员 / 保安 / 低调学生 / 被裁员的小职员开局。
- 鼓励：企业高管、法医、社工、运动员、电竞玩家、医生、刑警、记者。

## 4. 场景

- 觉醒场景不要在 "上班路上 + 便利店 + 下雨天"；这三要素同时出现 usage
  已超过 10 次。
- 斗法要有**现代后勤**：监控、社交媒体流出、维稳部门介入、医保流程。

## 5. 派系

- 政府异能管理司 ≠ "九州异能局"；优先挖掘真实可能存在的机构对标。
- 企业化修真（比如"灵能地产"、"灵气数据交易所"）是高潜力方向。
