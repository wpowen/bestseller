# 都市异能/权力逆袭调研方法论（v2.0）

调研都市异能题材时，依 **世界观隐秘层 → 能力体系 → 组织 → 人物 → 情节节奏 → 母题** 顺序挖掘。

## 1. 世界观（world_settings）

**目标**：贡献 ≥ 8 种"现代世界下超自然存在形式"，每种在社会结构层面有独立设定。**

- 探索：
  - 地下社会型（普通人不知道，隐秘门派在城市运行）
  - 灵气复苏型（全民知道，等级公开，社会分层）
  - 政府管控型（特务机构、异能警察、许可证制度）
  - 都市神话型（城市本身是神话结界，地铁/医院/酒店有隐藏层）
  - 进化型（物种进化导致异能出现，科学可解释）
- 避免：简单的"普通人不知道异能者存在"设定——这是最过度消费的框架。
- `content_json` 须含：`urban_layer_model / power_public_awareness / faction_hide_mechanism / unique_conflict_source`

## 2. 能力体系（power_systems）

**目标：≥ 10 种异能类型，其中 ≥ 4 种不是超速/超强/再生/读心。**

- 有深度的异能类型：
  - 信息流（读取过去/伪造记忆/预测概率）
  - 规则型（修改局部物理规则，如摩擦力、时间流速）
  - 编辑型（从现实删除/插入对象）
  - 情绪/场域型（操控情绪场、营造幻觉场）
  - 代谢型（超快进化/自我改写身体结构）
- 每条 entry 必含：`ability_type / activation_cost / social_impact / common_misuse`
- 所有能力必须有真实代价（体力/精神/时间/社会代价）

## 3. 组织（factions）

- 探索非家族类型：
  - 行业协会型（能力者认证机构，有官僚腐化逻辑）
  - 宗教伪装型（以宗教形式运作的秘密组织）
  - 政府机构型（科研局、特种部队，有内部政治）
  - 自发互助型（能力者地下网络，松散但有规则）
- 每个 entry 要有：`ideology / resource_base / internal_conflict / relationship_with_normals`

## 4. 人物（character_archetypes）

**禁止复用"废柴男主 + 秘书女配 + 土豪反派"三件套。**

- 调研非主流原型：
  - 职场精英型（已有地位但能力被压制）
  - 被迫参与型（本来只想普通活，被卷进秘密组织）
  - 双面型（白天正常人，夜晚隐秘执法者）
  - 复仇终局型（一切准备已做完，故事从爆发点开始）
  - 体制内破局型（异能者在官方体制内，从内部改变）
- 主角的核心竞争力不仅是"能力更强"，还有：判断力/信息/人脉/职业技能

## 5. 情节节奏（plot_patterns）

- 避免"打脸三件套"（羞辱 → 崛起 → 当众反击）作为唯一驱动。
- 探索：
  - 商战并轨（崛起的同时执行商业策略）
  - 组织斗争（内部清洗为主线，主角是棋局中的棋手而不是棋盘主宰）
  - 信息战（情报而非武力，线人/间谍/记忆操控）
- 每个阶段的崛起必须有人际代价：得到权力意味着失去某种关系

## 6. 对话设计（dialogue_styles）

- 职场/商场对话的潜台词：说的是业务，传达的是权力威胁
- 隐藏高手的语言：表面平凡，内行人能解读出信号——双层阅读
- 商战谈判：非直接强硬，而是"我知道你知道我知道"的博弈感

## 7. 真实参考（real_world_references）

- **秘密组织历史**：三合会、共济会、CIA黑历史——提供组织运作的真实逻辑参考
- **心理学**：权力动态（Milgram服从实验、社会角色理论）
- **中国道门现代传承**：民间术士、阴阳先生的社会地位，提供"现代秘术者"人设参考
- **职场权力结构**：组织行为学，权力不只是职位，还有信息/关系/专业权力
