# 穿书反派/重生女配/乙女游戏调研方法论

调研穿书/穿越反派题材时，以 **元小说框架 → 命运改写逻辑 → 反派重定义 → 情感真实化 → 代价机制** 推进。

## 1. 世界观（world_settings）

**目标**：建立 ≥ 5 种"穿书/穿越后的世界规则"，每种对"命运改变"的阻力不同。

- 命运阻力类型：
  - 剧情强制力（关键事件必须发生，可以调整参与者和结果）
  - 主角光环物理效果（原女主/男主自带buff，系统性优势）
  - 他人的预期（周围人对"反派"角色的固有偏见）
  - 知识有限（只记得看过的情节，记忆有空白）
- 每条设定须有"突破这个阻力需要付出什么代价"。

## 2. 人物（character_archetypes）

- 穿书女主不要立刻放弃"反派"属性——反派特质（锋芒、心机、自保意识）是她的差异化竞争力。
- 原女主不是天然坏人：她只是在执行原著逻辑，应有合理的独立目标。
- 各男主攻略路线要有各自的叙事逻辑，不能都莫名其妙爱上穿书主角。

## 3. 情节（plot_patterns）

- "改变命运"必须有三重代价：
  1. 消耗某种资源（时间、人际关系、身份信用）
  2. 触发某种连锁反应（改A则B变了）
  3. 制造信息差（穿书女主比其他人知道更多，但也有盲区）

## 4. 情感真实化（emotion_arcs）

- 元认知（知道自己在小说里）和情感投入是矛盾的——女主要有一个"开始真的在乎"的转折点，而不是始终冷静旁观。
- CP的建立必须超越"他不像书里写的那样"——要有真实的互动积累。
