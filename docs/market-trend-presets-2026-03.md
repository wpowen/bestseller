# 2026-03 网文热点题材与预设映射

更新时间：2026-03-20

这份文档只做一件事：把近阶段更容易起量、可读性更强、平台更愿意分发的题材，整理成 BestSeller 可以直接使用的预设方向。

## 结论

近阶段更值得优先尝试的，不是“任何题材都能靠 prompt 写好”，而是这几类已经被平台和读者反复验证过的方向：

- 末日囤货 / 末日规则 / 资源差生存
- 无 CP 大女主 / 强主体性女性成长
- 修仙 2.0 / 现代修真 / 高概念升级
- 民俗悬疑 / 规则怪谈 / 人性博弈
- 历史穿越经营 / 考据式制度推进
- 重生创业 / 现实事业翻盘

这些方向已经被映射为系统里的 `GenrePreset` 热门预设，并带有：

- `trend_score`
- `trend_summary`
- `trend_keywords`
- `trend_source_refs`

## 热点信号

### 1. 平台依然偏爱强钩子、强追读、强爽点

番茄作者专区的公开文章一直在强调两件事：

- 开篇必须尽快亮出主角优势、利益和危险
- 连载必须持续制造追读钩子和高密度推进

这意味着：末日、规则、资源差、身份翻盘这类能在几百字内立刻起势的题材，天然更适合当前平台节奏。

来源：

- [番茄作者专区：如何设计吸引读者的开篇](https://fanqienovel.com/writer/zone/article/7480087779494346776)
- [番茄作者专区：长篇写作中的节奏与持续追读](https://fanqienovel.com/writer/zone/article/7517213257795387416)

### 2. 2025 年底到 2026 年初，公开报道里出现了更明确的热点关键词

《中国新闻周刊》联合番茄小说发布的“2025 网文十大关键词”里，能直接转化为题材方向的包括：

- 修仙 2.0
- 无 CP 大女主
- 人性流
- 考据式穿越
- 东方美学

这批关键词说明市场不是简单地回到老题材，而是偏向“成熟题材 + 新包装 + 更鲜明的人群定位”。

来源：

- [番茄小说发布 2025 网文十大关键词](https://m.thepaper.cn/newsDetail_forward_30716745)

### 3. 官方行业报告也在支持这个判断

中国作协网络文学中心 2025 年的公开文章提到：

- 男频更重“设定创新”和主角成长引擎
- 女频更强调“女性主体意识”和“力量感”
- 题材正在向现实感、专业感、细分人群体验深入

这对应到产品层，就是：

- 男频不能只写“升级”，要写升级引擎为何新
- 女频不能只写“关系”，要写主体性和选择代价

来源：

- [中国作协：2025 网络文学观察](https://www.chinawriter.com.cn/n1/2025/0317/c404027-40440912.html)

### 4. 平台首页和分类结构也说明题材基本盘没有变

当前主流平台首页和分类仍然稳定覆盖：

- 玄幻 / 仙侠 / 科幻 / 都市 / 历史 / 悬疑
- 女频成长 / 古言 / 甜宠 / 复仇

这意味着“热点”不能脱离平台基本盘。真正有效的做法是：

- 在平台基本盘里找更容易起量的细分切口
- 而不是脱离平台读者去追空泛的新题材

## 已映射到系统的热点预设

当前已经映射为热点预设的方向：

- `apocalypse-supply`
- `rule-horror`
- `female-no-cp`
- `urban-xiuxian-2-0`
- `historical-research-travel`
- `human-nature-game`
- `folk-mystery`
- `female-growth-romance`

可以直接查看：

```bash
./scripts/run.sh writing-preset hot --limit 8
./scripts/run.sh writing-preset show female-no-cp --kind genre
./scripts/run.sh writing-preset show urban-xiuxian-2-0 --kind genre
```

## 产品决策

在 BestSeller 里，热点题材不直接替代基础题材预设，而是作为一个额外推荐层：

- 保留长期稳定的题材目录
- 在目录上方增加“当前更值得先试”的排序
- 在页面中显示热点说明、关键词和来源

这样做的好处是：

- 不会把系统绑死在短期热点上
- 仍然可以用长期题材结构做大长篇
- 选题时能优先看到当前成功概率更高的方向
