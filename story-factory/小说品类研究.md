# 小说品类评审研究

> 本文档是 BestSeller 品类感知评审系统的核心参考资料。
> 所有品类维度权重、规划校验逻辑和裁判提示词的设计依据均来源于此。
> 更新时间：2026-04

---

## 一、总论：为什么不同品类需要不同的评审标准

网文市场的核心矛盾是：**读者按品类选书，但大多数生成系统按通用标准写书。**

一部末日生存小说和一部都市言情小说，如果用同一把尺子来衡量，结果必然是：末日文被扣"情感深度不足"，言情文被扣"世界观单薄"。这不是质量问题，而是尺子错了。

品类评审的核心原则：

1. **读者承诺决定评审重心。** 每个品类对读者做出的核心承诺不同，评审必须围绕该承诺是否兑现来打分。升级文的承诺是"主角一定会变强"，悬疑文的承诺是"谜底一定有逻辑"，言情文的承诺是"感情线一定会推进"。承诺未兑现是致命缺陷，其余维度可以适度宽容。

2. **权重倾斜而非维度删减。** 所有品类都需要评审"节奏""冲突""角色""世界观""文笔"等基本维度，但不同品类对这些维度的权重分配截然不同。升级文的"力量体系一致性"权重远高于"散文诗意"，东方美学文的"意境营造"权重远高于"升级节奏"。

3. **弱项容忍有边界。** 品类允许某些维度分数偏低，但不意味着可以彻底忽视。悬疑文可以容忍"角色关系深度"偏低，但不能容忍"角色行为动机不合逻辑"——因为后者直接破坏推理链。

4. **品类信号是质量的外在指标。** 每个品类有一组"信号关键词"，它们不是模板，而是高质量作品中自然出现的文本特征。评审模型通过检测这些信号的密度和分布来辅助判断品质。

5. **挑战必须跟随主角成长进化。** 无论哪个品类，如果所有卷的冲突类型、对抗力量、挑战难度完全相同，叙事必然陷入重复。不同品类的进化模式不同，但"必须进化"是跨品类的铁律。

## 二、品类评审分类体系

系统将所有网文品类映射到 9 个评审分类（review category），每个分类对应一套独立的维度权重、规划校验规则和裁判提示词。分类的设计原则是：

- **按叙事引擎分类，而非按题材分类。** "仙侠"和"都市异能"题材不同，但叙事引擎都是"主角升级打怪"，因此归入同一个评审分类。
- **一个作品只匹配一个评审分类。** 如果一部作品同时具有升级和言情元素，系统根据主线叙事引擎决定分类，而非取交集。
- **`default` 是兜底而非推荐。** 无法明确归类的作品使用通用标准，但通用标准的区分度最低，应尽量避免。

### 9 个评审分类一览

| 分类 key | 中文名 | 叙事引擎 | 典型题材 |
|---|---|---|---|
| `action-progression` | 动作升级类 | 主角通过战斗/升级不断变强 | 玄幻、仙侠、都市异能、末日生存、科幻机甲 |
| `relationship-driven` | 关系情感类 | 核心关系的建立/深化/考验 | 言情、古言、甜宠、虐恋、耽美、百合 |
| `suspense-mystery` | 悬疑推理类 | 谜团的层层揭开与逻辑闭环 | 推理探案、规则怪谈、民俗诡事、惊悚 |
| `strategy-worldbuilding` | 策略世界观类 | 势力博弈与世界规则的运用 | 历史权谋、战争争霸、穿越经营、政治小说 |
| `esports-competition` | 电竞竞技类 | 比赛/对局/排名的持续攀升 | 电竞、游戏、无限流、副本系统流 |
| `female-growth-ncp` | 女性成长无CP类 | 女性主体性与个人价值的实现 | 大女主、宫斗、女帝、职场逆袭 |
| `base-building` | 基建经营类 | 从零开始的建设与资源积累 | 种田、基建、重生经商、领地经营 |
| `eastern-aesthetic` | 东方美学类 | 意境、哲思与文化沉浸 | 东方美学幻想、国风、水墨仙侠 |
| `default` | 通用 | 无明确叙事引擎偏向 | 未归类题材 |

## 三、各品类详细评审标准

### 3.1 action-progression（动作升级类）

#### 核心读者承诺

主角一定会变强，力量体系有逻辑，战斗有策略而非无脑碾压，升级节奏持续给读者"成长爽感"。读者来这个品类，买的是"看主角从弱到强的完整旅程"。

#### 评分维度权重倾斜原理

- **力量体系一致性**（权重极高）：升级文的核心资产。如果力量体系自相矛盾，整部作品的叙事地基崩塌。读者对体系 bug 的容忍度极低。
- **战斗节奏与策略感**（权重高）：战斗不能只是数值碾压，需要展示主角的智慧和成长。纯粹的"我比你强所以我赢"会让读者失去兴趣。
- **升级里程碑分布**（权重高）：升级节奏过密导致膨胀感丧失，过疏导致追读疲劳。理想分布是每 3-5 章一个小突破，每卷一个大突破。
- **对抗力量多元化**（权重中）：如果主角始终对抗同一类敌人，读者会审美疲劳。需要切换对抗维度（个体→组织→势力→规则→天道）。
- **散文诗意 / 抒情描写**（权重低）：升级文读者对文笔的期待是"干净利落"而非"词藻华丽"。过度抒情反而拖慢节奏。

#### 规划校验要点

- `power_tier_escalation`：世界观中力量体系必须定义至少 3 个层级；主角在不同卷的力量层级必须有递进，不能始终停留在同一层级。
- `antagonist_evolution`：对抗力量必须包含 2 种以上的 `force_type`（character/faction/environment/internal/systemic）；不同卷的 `primary_force_name` 必须有轮替。
- `conflict_phase_variety`：4 卷以上的作品必须使用至少 3 种冲突阶段类型（survival/political_intrigue/betrayal/faction_war/existential_threat 等）。

#### 裁判模型审核重点

- 战斗场景是否展示了策略和变化，而非模式化的"蓄力→释放→获胜"。
- 升级后主角的行为模式是否发生了可感知的变化（视野、决策、对手级别）。
- 力量体系的规则是否在战斗中被一致地遵守，有无违反已建立规则的情况。
- 每章结尾是否留下了与升级/战斗相关的钩子（新威胁、新能力线索、新对手）。

#### 弱项容忍范围

- **情感深度**：可以偏低（0.5-0.6 即可接受），升级文不需要复杂的感情纠葛。
- **哲学深度**：可以偏低，读者不期待深刻的人生思考。
- **诗意描写**：可以偏低，简洁有力的战斗描写优于华丽辞藻。
- **不可容忍**：力量体系矛盾（一票否决级别）、升级节奏完全停滞超过连续 3 章、战斗无策略变化。

#### 品类信号关键词

战斗与力量类：突破、晋级、领悟、气劲、灵力、真气、吞噬、压制、碾压、对峙、绝招、秘技、禁术
升级节奏类：修炼、闭关、顿悟、瓶颈、契机、蜕变、进阶、淬体、凝丹
体系规则类：位阶、层级、境界、法则、规则、天道、限制、代价、反噬
追读钩子类：隐藏、觉醒、封印、血脉、传承、秘境、遗迹

#### 常见品质陷阱

1. **数值膨胀失控**：力量数字越写越大，但读者对数字已经麻木，失去成长爽感。应该用"质变"代替"量变"。
2. **战斗描写模板化**：每场战斗都是"对峙→试探→大招→反转→获胜"的固定流程。需要变换战斗环境、条件、目标。
3. **配角工具化**：配角只在主角需要被救或需要观众时才出现，缺乏自身的行为逻辑。
4. **升级理由随意化**：突然"感悟"或"被激发"导致突破，缺乏铺垫和因果链。每次突破都应该有明确的触发条件和代价。
5. **力量体系后期崩塌**：前期设定严谨，后期为了推进剧情开始违反自己建立的规则。

### 3.2 relationship-driven（关系情感类）

#### 核心读者承诺

核心关系一定会推进，情感互动有层次和变化，角色之间的化学反应真实可感。读者来这个品类，买的是"看两个人（或一组关系）如何从A走到B的完整情感旅程"。

#### 评分维度权重倾斜原理

- **情感真实性与层次**（权重极高）：情感描写是否让人信服。"他突然爱上了她"不可接受，必须有充分的心理铺垫和情感递进。
- **关系里程碑分布**（权重高）：关系发展不能停滞。每卷必须有至少一个关系质变点（信任建立/误解产生/靠近/选择/和解）。
- **角色独立性**（权重高）：角色不能沦为关系的附庸。每个核心角色都需要有独立的行为动机和人生目标。
- **对话质量**（权重高）：情感类作品中对话承载了大量的情感信息。对话必须符合角色性格，并推进关系发展。
- **力量体系复杂度**（权重低）：如果有力量体系（如仙侠言情），体系本身的复杂度要求较低，但体系规则不能矛盾。

#### 规划校验要点

- `relationship_milestone_progression`：卷目标和卷主题中必须包含关系发展关键词（信任、误解、靠近、选择、并肩、质变等）。至少半数以上的卷应体现关系线推进。
- `emotional_arc_explicit`：每卷必须有明确的情感目标（volume_goal 或 volume_theme），不允许出现"空心卷"。

#### 裁判模型审核重点

- 核心关系在本章是否发生了可感知的微变化（不一定是大转折，但不能原地踏步）。
- 角色的情感反应是否与其性格设定一致，有无"为了剧情需要突然变心"的情况。
- 对话是否承载了情感信息，还是只有功能性的情节推进对话。
- 情感场景是否调动了身体语言、环境暗示、内心独白等多重表达手段。

#### 弱项容忍范围

- **世界观深度**：可以偏低（0.4-0.6 即可接受），言情读者不要求宏大世界观。
- **战斗/动作场景**：可以偏低甚至缺失，除非是以战斗为背景的言情。
- **升级体系**：可以完全没有，不扣分。
- **不可容忍**：核心关系连续 2 卷以上无任何实质推进、角色行为严重违反已建立的性格设定、情感转折缺乏铺垫。

#### 品类信号关键词

情感互动类：心跳、呼吸、靠近、后退、眼神、沉默、温度、触碰、距离、手指
关系变化类：信任、试探、误解、和解、选择、放手、守护、承诺、牺牲、并肩
内心独白类：不自觉、忍不住、明知道、却还是、第一次、从未、习惯、在意、心软
氛围营造类：月光、雨声、夜晚、窗台、屋檐、街角、背影、余温、风

#### 常见品质陷阱

1. **误解驱动变拖延驱动**：用误解制造冲突是常见手法，但如果同一个误解反复出现且可以一句话解决，读者会失去耐心。
2. **角色降智服务剧情**：让聪明角色突然变蠢来制造情感冲突。应该用价值观差异、信息差等合理因素驱动冲突。
3. **配角工具化严重**：闺蜜/兄弟只在需要倾诉时出场，情敌只负责搅局。配角应有独立人格。
4. **情感描写模板化**：每次心动都是"心跳加速"，每次伤心都是"眼眶泛红"。需要更多元的身体和心理表达方式。
5. **关系进展不均匀**：前期极慢，读者流失；后期赶工，情感质变缺乏铺垫。需要在规划阶段就均匀分配关系里程碑。

### 3.3 suspense-mystery（悬疑推理类）

#### 核心读者承诺

谜底一定有逻辑，线索一定有回收，真相一定比表面更复杂。读者来这个品类，买的是"和主角一起推理、被合理反转震撼的智识体验"。

#### 评分维度权重倾斜原理

- **逻辑自洽性**（权重极高）：推理的生命线。任何逻辑漏洞都会让读者感到被愚弄，直接导致弃书。
- **线索链完整性**（权重极高）：所有关键线索必须在揭秘前至少出现过一次（公平推理原则），读者有机会自己推理出真相。
- **信息释放节奏**（权重高）：信息不能一次性倾泻，也不能长期不给新线索。理想节奏是每章推进至少一条线索。
- **氛围营造能力**（权重中）：悬疑/恐怖氛围是加分项，但不能替代逻辑。纯靠氛围吓人的作品在推理读者中评价偏低。
- **角色关系深度**（权重低）：推理文的核心不是人物关系，但角色行为动机必须合逻辑。

#### 规划校验要点

- `clue_chain_exists`：`key_reveals` 必须分布在至少 2 个卷中，不能所有揭示集中在一卷。
- `misdirection_planned`：`foreshadowing_planted` 必须在至少 2 个卷中设置伏笔，为红鲱鱼和真正线索提供布局空间。
- `information_escalation`：关键揭示不能超过 80% 集中在某一卷，信息释放必须均匀分布。

#### 裁判模型审核重点

- 本章是否推进了线索链（发现新线索 / 排除错误方向 / 深化已有线索 / 制造新疑问）。
- 已揭示的线索之间是否存在逻辑矛盾。
- 角色的推理过程是否展示了思维链（让读者跟着推理），而非直接给出结论。
- 红鲱鱼是否有合理存在的理由，而非纯粹的误导废笔。
- 恐怖/悬疑氛围是否有节奏变化（不能全程高压，需要喘息再收紧）。

#### 弱项容忍范围

- **角色关系深度**：可以偏低（0.4-0.5 即可接受），推理读者不要求复杂感情线。
- **战斗/动作场景**：可以偏低或没有。
- **升级体系**：不需要。
- **散文诗意**：可以偏低，推理文风格通常偏冷硬或纪实。
- **不可容忍**：逻辑漏洞（一票否决）、谜底靠巧合或超自然强行解释、关键线索无铺垫直接出现、规则怪谈的规则没有内在逻辑。

#### 品类信号关键词

线索与推理类：线索、证据、痕迹、矛盾、疑点、排除、推断、假设、验证、动机
悬疑氛围类：异常、诡异、寂静、阴影、不对劲、违和感、第六感、直觉、警觉
反转与揭秘类：真相、反转、揭露、原来、果然、不可能、盲点、误导、隐藏
恐怖元素类：规则、禁忌、不可、禁止、代价、诅咒、消失、不能回头

#### 常见品质陷阱

1. **线索凭空出现**：关键证据在需要时才第一次出现，读者完全没有机会提前推理。必须遵守"公平推理"原则。
2. **万能侦探综合症**：主角无所不知，每次都能一眼看穿。应该让主角也会走弯路、被误导。
3. **氛围掩盖逻辑缺失**：用大量恐怖描写来掩盖推理链的薄弱。氛围是锦上添花，不是核心。
4. **规则怪谈的规则不自洽**：规则体系前后矛盾，或者规则的存在只为恐吓而无内在逻辑。
5. **信息倾泻式揭秘**：在最后一章用大段独白一次性解释所有谜团，读者来不及消化也无法验证逻辑。

### 3.4 strategy-worldbuilding（策略世界观类）

#### 核心读者承诺

世界观有深度和逻辑，势力博弈有策略，主角的谋略和决策是推动剧情的核心引擎。读者来这个品类，买的是"看一个复杂世界中的战略博弈与权力运作"。

#### 评分维度权重倾斜原理

- **世界观完整性与深度**（权重极高）：这个品类的核心资产。世界规则、势力分布、资源体系、制度运作都需要有足够的深度和一致性。
- **策略合理性**（权重极高）：主角的决策必须在已建立的世界规则下合理可行，不能靠作者"偏心"取胜。
- **势力格局动态变化**（权重高）：势力平衡必须随剧情推进而变化，不能始终是"主角 vs 一个大反派"的静态格局。
- **信息差运用**（权重中）：好的策略叙事善于利用角色之间的信息差来制造戏剧张力。
- **情感深度**（权重低-中）：策略文中情感不是核心，但关键角色的忠诚/背叛选择需要有情感根基。

#### 规划校验要点

- `faction_progression`：不同卷的 `primary_force_name` 必须有变化，不能始终对抗同一个势力。
- `worldbuilding_depth_check`：世界观设定中 `rules/world_rules` 至少 3 条，`locations` 至少 2 个。
- `conflict_phase_variety`：冲突阶段必须多元化，体现战略格局的演变（生存→政治→背叛→势力战→终极威胁）。

#### 裁判模型审核重点

- 主角的决策是否在已建立的世界规则和信息范围内合理，有无"开上帝视角"或"作者偏心"。
- 势力格局在本章是否发生了可感知的变化（哪怕是微小的力量平衡偏移）。
- 世界规则是否被一致遵守，有无为了推进剧情而违反规则的情况。
- 配角/对手的决策是否也有其自身逻辑，而非为了衬托主角而刻意降智。

#### 弱项容忍范围

- **散文诗意**：可以偏低，策略文更注重信息密度和逻辑清晰。
- **浪漫情感**：可以偏低（0.3-0.5），除非是以宫斗/宅斗为核心的策略文。
- **战斗描写细节**：可以偏低，策略文的"战斗"更多是运筹帷幄而非贴身肉搏。
- **不可容忍**：世界规则自相矛盾、主角靠"运气"而非"策略"获胜、所有对手都是弱智。

#### 品类信号关键词

势力与权谋类：朝堂、势力、阵营、同盟、背叛、拉拢、制衡、布局、下棋、博弈
策略与决策类：谋划、算计、反间、信息、情报、筹码、利益、让步、交换、牺牲
世界规则类：制度、律法、规矩、传统、禁令、资源、供给、税赋、封地、领地
格局变化类：崛起、衰落、吞并、独立、联盟、割据、统一、变法、改革

#### 常见品质陷阱

1. **主角全知全能**：主角对所有势力的底细了如指掌，博弈毫无悬念。应该让主角也有信息盲区。
2. **对手集体降智**：为了让主角的策略成功，所有对手都犯低级错误。对手应该是聪明的，主角赢在信息优势或更好的执行力。
3. **世界观展示变百科全书**：用大段说明文介绍世界设定，而非通过角色的体验和决策来展示。
4. **势力格局静态化**：从第一卷到最后一卷，势力分布没有本质变化。应该像棋局一样不断重组。
5. **谋略与执行脱节**：花大量篇幅描写谋划过程，但执行阶段一笔带过，读者看不到策略如何落地。

### 3.5 esports-competition（电竞竞技类）

#### 核心读者承诺

比赛一定有看点，技术成长可感知，竞技氛围真实热血。读者来这个品类，买的是"和主角一起从菜鸟打到巅峰的热血竞技旅程"。

#### 评分维度权重倾斜原理

- **比赛场景的专业性与紧张感**（权重极高）：竞技文的核心场景。比赛描写必须让读者感受到紧张、刺激和战术变化，而非流水账式的"A打了B，B反击了A"。
- **技术/战术成长弧线**（权重高）：主角的成长必须在竞技层面可感知——从操作到意识到领导力的递进。
- **团队动态**（权重高）：大多数电竞题材涉及团队。队友之间的化学反应、磨合冲突、默契培养是重要的叙事维度。
- **竞技规则与体系**（权重中）：游戏/竞技规则需要清晰，但不需要真实游戏的完整复杂度。
- **世界观深度**（权重低）：电竞文的"世界"通常是现实世界+游戏内世界，不需要宏大的奇幻设定。

#### 规划校验要点

- 每卷应有至少一场关键比赛/对局作为高潮点。
- 主角的竞技水平在不同卷之间必须有可感知的提升。
- 对手的实力和风格需要多样化，不能每次都是同一种打法的敌人。
- 团队成员至少 2 名应有独立的性格特征和角色弧线。

#### 裁判模型审核重点

- 比赛场景是否有起伏（先落后再反超 / 关键时刻的决策 / 意想不到的战术），而非一边倒。
- 技术/战术描写是否具体到读者能"看到画面"，而非抽象概括。
- 非比赛章节是否在推进角色成长或团队建设，而非纯粹的日常水文。
- 主角的获胜是否有充分的技术/战术基础，而非"主角光环"。

#### 弱项容忍范围

- **世界观深度**：可以偏低（0.3-0.5），电竞文不需要宏大世界观。
- **哲学深度**：可以偏低，竞技文更注重热血和成长。
- **散文诗意**：可以偏低，简洁有力的比赛解说风格更适合。
- **不可容忍**：比赛结果全靠运气或主角光环、技术描写错误且不自洽、团队成员完全没有存在感。

#### 品类信号关键词

竞技操作类：操作、走位、预判、反应、手速、极限、失误、完美、闪现、技能
战术思维类：策略、阵容、节奏、视野、资源、对线、团战、切入、牵制、佯攻
团队协作类：配合、信任、默契、指挥、沟通、补位、掩护、开团、撤退
成长竞技类：排位、晋级、决赛、冠军、巅峰、挑战、突破、瓶颈、训练、复盘

#### 常见品质陷阱

1. **比赛描写流水账化**：机械地记录每一步操作，缺乏节奏起伏和情感投入。比赛描写应该像体育解说一样有重点、有高光。
2. **技术描写过于真实或过于虚假**：完全照搬真实游戏术语让非玩家读者看不懂；完全虚构又让核心玩家群体出戏。需要平衡。
3. **主角一人carry全队**：队友完全沦为背景板，违背团队竞技的基本逻辑。
4. **训练/日常占比过大**：非比赛内容超过 60% 会导致节奏拖沓。需要合理安排比赛与训练/日常的比例。
5. **对手脸谱化**：对手只有"嚣张→被打脸"一种模式。应该有值得尊敬的对手、有独特风格的对手。

### 3.6 female-growth-ncp（女性成长无CP类）

#### 核心读者承诺

女主角一定会通过自身的智慧、能力和选择实现价值跃升，成长路径以个人主体性为核心而非依赖感情线。读者来这个品类，买的是"看一个女性角色如何靠自己的力量改写命运"。

#### 评分维度权重倾斜原理

- **女主角主体性与选择代价**（权重极高）：女主角的每一个关键决策必须是她自己做出的，且要承受真实的代价。不能靠"白马王子拯救"或"金手指无脑开挂"。
- **权力/能力成长弧线**（权重高）：成长必须可感知且有逻辑。从被动到主动、从边缘到核心、从无权到有权的过程是核心叙事骨架。
- **人际博弈与心理深度**（权重高）：宫斗/职场/权力场中的人心博弈是重要看点。角色之间的算计、结盟、背叛需要有心理层面的深度。
- **男性角色的独立性**（权重中）：男性角色不能只是"工具人"或"被攻略对象"。他们应该有独立的目标和逻辑。
- **浪漫情感**（权重低）：无 CP 不意味着完全没有情感线索，但情感不能成为叙事主线。

#### 规划校验要点

- 女主角在每卷的 volume_goal 中必须有明确的个人能力/权力目标（而非关系目标）。
- 至少每隔一卷，女主角面对的对手/挑战的规模和维度应该升级。
- 女主角的成功不能全部依赖单一的"金手指"，需要展示多维能力。

#### 裁判模型审核重点

- 女主角的关键决策是否由她自己主导（而非被动接受他人安排或被推着走）。
- 成功是否伴随着真实的代价和取舍（"两全其美"的结局不可信）。
- 人际博弈中是否展示了心理层面的交锋（而非只有表面的对话冲突）。
- 女主角是否有脆弱和犹豫的时刻（完美角色缺乏感染力），但脆弱后必须自己站起来。

#### 弱项容忍范围

- **浪漫情感**：可以很低（0.2-0.4），品类定位就是无 CP。
- **战斗/动作场景**：可以偏低，成长不一定通过武力实现。
- **力量体系复杂度**：如果有的话可以简化。
- **不可容忍**：女主角的关键成功依赖男性角色的帮助/保护、成长没有代价（无脑开挂）、所有女性配角都是反派/工具人。

#### 品类信号关键词

主体性与选择类：选择、代价、赌注、取舍、底线、决定、不退、站住、凭什么、我要
权力与成长类：权力、位置、筹码、翻盘、破局、晋升、扎根、势力、根基
心理博弈类：试探、算计、结盟、背叛、示弱、示强、虚虚实实、城府、洞察
女性力量类：清醒、冷静、隐忍、爆发、不求人、自己的路、姿态、骨气

#### 常见品质陷阱

1. **假无 CP 真恋爱**：名义上无 CP，但叙事重心不自觉地滑向男女关系线。需要在规划阶段就明确主线是个人成长而非感情。
2. **爽文化陷阱**：女主角"打脸"成瘾，所有对手都是脑残反派。应该设置有实力有智慧的对手。
3. **"圣母"或"黑莲花"极端**：角色要么完美无私到不真实，要么阴暗算计到缺乏共情。好的角色是复杂的。
4. **成长路径单一化**：只通过"惩罚坏人"来体现成长。真正的成长是多维度的——能力、视野、格局、自我认知。
5. **所有男性角色都是NPC**：为了突出女主而把所有男性角色写成背景板，反而显得世界不真实。

### 3.7 base-building（基建经营类）

#### 核心读者承诺

从零到一的建设过程有细节有满足感，资源积累和管理有逻辑，"种田"的过程本身就是最大的看点。读者来这个品类，买的是"看一个体系从无到有、从小到大的全过程满足感"。

#### 评分维度权重倾斜原理

- **建设细节的真实感与满足感**（权重极高）：经营文的核心爽点。读者要的是"今天修了路、明天开了矿、后天建了城墙"的积累快感。细节必须具体且有因果逻辑。
- **资源体系的自洽性**（权重高）：资源从哪来、怎么转化、有什么限制——这套体系必须自洽。"随便挖到宝藏"不可接受。
- **成长的可视化**（权重高）：读者需要能"看到"基业的规模变化。从一个帐篷到一座城市，每个阶段的变化要清晰。
- **外部威胁与内部管理的平衡**（权重中）：纯建设会缺乏张力，需要适度的外部威胁（兽潮、敌国、天灾）和内部管理挑战（人才、叛乱、资源危机）。
- **情感深度**（权重低-中）：基建文中的人际关系更多是"合作伙伴"而非"恋人"关系。

#### 规划校验要点

- 每卷的 volume_goal 应包含具体的建设/经营目标（扩张领地、开辟新资源、建设新设施等）。
- 资源体系应在世界观设定中明确定义，且各卷中遵守一致的资源规则。
- 外部威胁的规模应随基业规模增长而升级。

#### 裁判模型审核重点

- 建设过程是否有具体细节（材料从哪来、怎么建造、花了多长时间），而非"时间一晃就建好了"。
- 资源体系是否自洽（消耗与获取平衡、技术树有合理的先后顺序）。
- 外部威胁是否与当前基业规模匹配（太弱无张力，太强不合理）。
- 主角的经营决策是否有策略性（取舍、优先级、风险管理），而非"什么都建"。

#### 弱项容忍范围

- **战斗/动作场景**：可以偏低（0.3-0.5），基建文的战斗更多是防御/保护，不需要华丽的战斗描写。
- **散文诗意**：可以偏低，实用主义的文风更符合品类气质。
- **角色关系深度**：可以偏低，功能性关系（师徒、搭档、下属）的描写优先于情感关系。
- **不可容忍**：资源凭空出现（破坏积累感）、建设过程被跳过（读者来看的就是过程）、没有任何外部威胁导致毫无张力。

#### 品类信号关键词

建设经营类：建造、修建、扩建、开垦、开采、冶炼、铸造、规划、蓝图、工坊
资源管理类：资源、物资、储备、产量、消耗、交易、采购、运输、仓储、盈余
成长阶段类：雏形、初具规模、繁荣、扩张、壮大、自给自足、产业链、升级
外部挑战类：兽潮、入侵、天灾、封锁、资源争夺、领地争端

#### 常见品质陷阱

1. **建设没有限制**：主角想建什么就建什么，资源无限、技术无限。缺乏限制就缺乏戏剧张力。
2. **跳过建设过程**："三个月后，城池已初具规模"——读者来看的就是这三个月的过程，不能跳过。
3. **技术树不合理**：还没有基本冶铁技术就直接造蒸汽机。技术发展必须有合理的先后顺序。
4. **只有建设没有管理**：建好后完全不提运营和管理问题。城市需要治安、粮食供应、人口管理等。
5. **外部威胁过于频繁或过于稀少**：频繁导致建设永远被打断；稀少导致缺乏紧迫感。需要合理的节奏。

### 3.8 eastern-aesthetic（东方美学类）

#### 核心读者承诺

意境之美贯穿全书，哲学思辨有深度，文化元素浑然天成而非生硬堆砌。读者来这个品类，买的是"在一个充满东方美学的世界中获得精神层面的体验和感悟"。

#### 评分维度权重倾斜原理

- **意境营造能力**（权重极高）：东方美学文的核心资产。每个重要场景都应该有"让读者停下来感受"的意境描写，但必须服务于叙事而非独立存在。
- **哲学深度与表达方式**（权重极高）：哲学思辨必须通过角色的选择和困境来呈现，而非直接灌输。"道"的体悟要自然地融入角色的成长弧线。
- **文化元素的融入度**（权重高）：诗词、典故、器物、仪式等文化元素必须与情节有机结合，不能变成知识科普或百科展示。
- **留白技巧**（权重高）：不说满、不写尽是东方美学的核心技法。过度解释和填满每一个细节是此品类的大忌。
- **节奏与画面感**（权重中）：叙事节奏要像中国画的"疏密"——留白章节后接密集冲突，张弛有度。

#### 规划校验要点

- 每卷应有一个核心意象或哲学命题作为主题（残缺、轮回、空与色、守与破等）。
- 世界观美学体系应统一：选定一种主导美学风格（水墨/青绿/枯山水/敦煌），贯穿全书。
- 冲突设计应有"美学张力"——美好事物的毁灭、理想与现实的撕裂。

#### 裁判模型审核重点

- 美学描写是否与情节/角色情感有机关联，还是纯粹的"美文"堆砌。
- 哲学/文化元素是否自然融入，读者是否需要额外知识才能理解。
- 叙事节奏是否有留白——是否每段都塞满了内容，还是有呼吸的空间。
- 角色弧线是否在美学表达中推进——本章结束时角色的认知/心境是否发生了变化。
- 高潮场景是否实现了"美学与戏剧张力的统一"——最美的场景承载最激烈的冲突。

#### 弱项容忍范围

- **升级节奏**：可以偏低（0.3-0.5），东方美学文的成长更多是"悟道"而非"升级"。
- **战斗频率/密度**：可以偏低，战斗更注重美感和意境而非密度。
- **信息密度**：可以偏低，留白本身就是品类特征。
- **不可容忍**：美学描写完全脱离叙事（纯粹炫技）、哲学说教化（直接灌输道理而非通过故事呈现）、文化元素堆砌但不理解其含义。

#### 品类信号关键词

意境营造类：水墨、青绿、留白、意境、空灵、清寂、幽远、恬淡、苍茫
自然意象类：枯梅、残月、流水、山岚、松竹、云雾、晚霞、秋雨、寒鸦
哲学思辨类：道、空、虚、实、破、立、轮回、因果、执念、放下、顿悟
文化元素类：琴棋书画、茶道、香道、剑道、花道、诗词、典故、祝祷、仪轨

#### 常见品质陷阱

1. **辞藻堆砌代替意境**：堆砌华丽形容词但场景没有画面感。一个精准的动词胜过三个华丽的形容词。
2. **哲学说教化**：作者借角色之口直接讲道理，变成议论文。哲学应该通过角色的选择和代价来呈现。
3. **美学与叙事割裂**：美的段落和推进剧情的段落泾渭分明，读者可以跳过"美文"不影响理解剧情。美学必须服务于叙事。
4. **文化元素"掉书袋"**：大量引用诗词典故但不理解其深层含义，变成知识展示而非故事融入。
5. **只有意境没有冲突**：沉浸在优美的描写中，忘记了故事需要冲突和张力来推动。

### 3.9 default（通用）

#### 核心读者承诺

提供完整、连贯、有吸引力的叙事体验。通用标准适用于无法明确归入以上 8 个品类的作品。

#### 评分维度权重倾斜原理

通用分类对所有维度均匀分配权重，不做特别倾斜。这意味着：
- 每个维度都有基本的达标要求。
- 没有维度可以特别弱，也没有维度会获得特别高的权重加成。
- 通用标准的区分度最低——它无法识别品类特有的优势和劣势。

#### 规划校验要点

仅运行通用校验项：
- `volume_goals_distinct`：卷目标不能重复。
- `challenge_evolution`：冲突阶段和对抗力量不能完全相同。
- `foreshadowing_balance`：伏笔需要有回收。
- `hooks`：非末卷必须有续读钩子。
- `volume_themes`：每卷必须有明确主题。

#### 裁判模型审核重点

- 故事是否有清晰的主线和合理的节奏。
- 角色是否有明确的动机和一致的行为逻辑。
- 冲突是否有递进和变化。
- 每章是否在推进叙事。

#### 弱项容忍范围

通用分类对所有维度的容忍下限均为 0.5，没有特别的宽容项。

#### 品类信号关键词

无特定品类信号。通用模型关注的是叙事基本面：冲突、角色、节奏、钩子、主题。

#### 常见品质陷阱

1. **品类不明确导致读者错配**：读者不知道这是什么类型的书，期待与实际内容不匹配导致差评。建议明确品类归属。
2. **试图讨好所有人**：什么元素都有一点但什么都不深入，结果是所有维度都平庸。
3. **叙事引擎缺失**：没有一个核心的叙事引擎来驱动故事前进，导致剧情散漫。

## 四、平台差异与品类评审的交叉影响

不同发布平台对内容有不同的偏好和要求，这些偏好会叠加在品类评审标准之上，形成"品类 x 平台"的复合评审矩阵。

### 4.1 番茄小说（Fanqie）

**平台特征：** 免费阅读模式，以广告变现为核心，极度重视追读率和完读率。

**对品类评审的影响：**
- **所有品类**的"开篇钩子"权重提高。番茄要求第一章内必须亮出主角差异化优势、当前利益和即时危险。
- **升级类/动作类**天然适配番茄节奏，但需要提高"短回报密度"——每 1-2 章就要有一个小爽点。
- **策略类/美学类**在番茄上需要适度加快节奏，减少铺垫和留白，否则追读率会下降。
- **悬疑类**适配度高，但每章的悬念钩子必须更加强烈直接。
- 章节字数通常控制在 2000-3000 字左右，信息密度高于其他平台。

### 4.2 起点中文网（Qidian）

**平台特征：** 付费订阅模式，读者付费意愿与内容深度正相关，允许更完整的设定展开。

**对品类评审的影响：**
- **策略类/世界观类**在起点可以有更大的展开空间，读者接受更长的设定铺垫。
- **升级类**是起点的核心品类，力量体系的完整性和创新性权重更高。
- 读者对"注水"敏感度更高，因为每一章都要花钱。章节内容密度要求高于免费平台。
- 卷级结构和长线叙事受到更多重视，"地图扩张"和"位阶突破"是起点读者的核心期待。

### 4.3 晋江文学城（Jinjiang / JJWXC）

**平台特征：** 女频为主的付费平台，读者群以女性为主，对角色塑造和情感表达要求极高。

**对品类评审的影响：**
- **关系情感类**的情感真实性权重进一步提高。晋江读者对"虚假情感"的容忍度极低。
- **女性成长类**在晋江有天然受众基础，但对女主角的"讨喜度"有更高期待。
- **东方美学类**与晋江的审美传统高度契合，文笔和意境权重可以进一步提高。
- 晋江读者重视"角色魅力"维度——角色不需要完美，但必须有让人记住的特质。
- 评论区文化活跃，读者反馈对创作方向有较大影响。

### 4.4 Amazon KU（Kindle Unlimited）

**平台特征：** 英文市场全球分发，按阅读页数分成，读者行为偏向快速翻页和系列消费。

**对品类评审的影响：**
- 开篇节奏要求比中文网文平台更快——英文市场的"试读门槛"更低但"弃书速度"更快。
- **升级类 / Progression Fantasy** 在 KU 有成熟的受众，但英文读者对"硬体系"（Hard Magic System）的要求更严格。
- 翻译/跨语言项目需要关注文化适配——某些在中文语境中自然的表达在英文中可能显得突兀。
- 系列化和系列一致性权重高，KU 读者偏好可预期的系列体验。

### 4.5 Royal Road

**平台特征：** 英文 Web Fiction 平台，以 LitRPG 和 Progression Fantasy 为主，社区驱动。

**对品类评审的影响：**
- **升级类**是核心品类，但 Royal Road 读者对"数值体系"（Status Window / Character Sheet）有更明确的期待。
- 社区评价机制（rating/review）对早期章节质量极度敏感，开篇 5 章的评审权重应特别提高。
- 读者对"稳定更新"的期待很高，章节质量的方差（忽高忽低）比平均水平低更伤害评价。
- **悬疑类 / 策略类**在 Royal Road 的受众较小，但核心读者的质量要求更高。

## 五、挑战进化原则

**核心铁律：挑战必须跟随主角的成长而进化。**

这条原则跨越所有品类，但进化的模式因品类而异。违反这条原则的作品，无论哪个品类，都会出现"叙事停滞感"——读者觉得"这一卷和上一卷好像差不多"。

### 5.1 进化模式总览

| 品类 | 进化维度 | 典型进化路径 |
|---|---|---|
| 动作升级类 | 力量层级 + 对手规模 | 小怪→精英怪→Boss→势力→规则→天道 |
| 关系情感类 | 情感深度 + 关系复杂度 | 陌生→试探→靠近→考验→质变→并肩 |
| 悬疑推理类 | 真相层级 + 信息复杂度 | 表象→一层真相→更深谜团→认知颠覆→终极揭示 |
| 策略世界观类 | 博弈规模 + 政治维度 | 个人→家族→地方→国家→文明 |
| 电竞竞技类 | 竞技级别 + 战术维度 | 新手→排位→联赛→全国→世界→传奇 |
| 女性成长类 | 权力半径 + 决策复杂度 | 求生→站稳→扩张→博弈→格局→自由 |
| 基建经营类 | 经营规模 + 管理复杂度 | 帐篷→村落→城镇→城市→区域→文明 |
| 东方美学类 | 感悟深度 + 哲学维度 | 初见之惑→技艺精进→道心考验→破碎重塑→大道至简 |

### 5.2 进化的三条子原则

**原则一：量变到质变。** 进化不是简单地"敌人更强了"。到了某个阶段，冲突的性质本身应该发生变化。升级文中从"打赢某个人"变成"瓦解一个势力"；悬疑文中从"解开一个案件"变成"揭示一个世界观真相"。

**原则二：旧能力面对新挑战。** 进化不意味着旧能力完全失效。好的进化是让主角用旧能力面对新类型的挑战，被迫拓展能力边界。升级文中学会火系法术的主角面对水底世界；言情文中学会信任的角色面对必须保守秘密的困境。

**原则三：进化带来新的弱点。** 每次进化都应该暴露新的问题。升级后的主角面对更大的责任和更复杂的敌人；权力扩张后的女主面对更多的政治算计。进化与新弱点并行，叙事张力才能持续。

### 5.3 系统中的实现

在 BestSeller 的 `plan_judge` 中，挑战进化原则通过以下校验实现：

- `_check_challenge_evolution`：检查所有卷的 `conflict_phase` 和 `primary_force_name` 是否有变化。
- `_check_volume_goals_distinct`：确保每卷的 `volume_goal` 不重复。
- 品类专属检查（如 `power_tier_escalation` 检查力量层级递进，`faction_progression` 检查势力轮替）。

这些检查共同保证：无论哪个品类，叙事都在向前推进，挑战都在跟随主角成长而进化。
